A modern and responsive **shopping cart web application** built with React.  
This project showcases core e-commerce functionality and clean state management, making it a strong addition to a frontend portfolio.

---

## 🚀 Live Demo

👉 https://shopping-cart-mockk.netlify.app/

---

## ✨ Key Features

- 🛍️ Product listing page  
- ➕ Add items to cart  
- 🔄 Increase / decrease quantity  
- ❌ Remove items instantly  
- 💰 Dynamic total price calculation  
- 🔗 Multi-page navigation (Shop / Cart)  
- 📱 Fully responsive design  

---

## 🧑‍💻 Tech Stack

- **React** – Component-based UI  
- **React Router** – Navigation & routing  
- **JavaScript (ES6+)** – Application logic  
- **CSS Modules** – Scoped and maintainable styles  
- **Vite** – Fast development environment

## ⚙️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/cebotaritigran/shopping-cart-mock.git
cd shopping-cart-mock
```

### 2. Install dependencies

```
npm install
```

### 3. Run the app

```
npm run dev
```
